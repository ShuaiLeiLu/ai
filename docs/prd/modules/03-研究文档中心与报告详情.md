# 模块 PRD：研究文档中心与报告详情

## 1. 模块定位

研究文档中心负责承接 AI 研究员输出的结构化研究成果，是平台从“生成能力”走向“消费价值”的核心展示模块。文档详情页不仅承载报告阅读，还承载步骤可视化、风险提示、评论互动和继续追问能力。

## 2. 模块目标

- 让研究结果以高可读性、高结构化的方式被消费。
- 强化“AI 不是只会聊天，而是能产出完整研究报告”的用户认知。
- 支持用户从阅读进入下一步追问、讨论、分享和沉淀。
- 形成平台内容资产池，为社区、排行榜与推荐提供内容供给。

## 3. 页面组成

### 3.1 文档中心列表

- 热门文档
- 精选文档
- 公开研究员产出内容
- 文档卡片摘要
- 查看全部入口

### 3.2 文档详情页

- 标题区
- 作者与来源区
- 任务进度 / 计划步骤区
- 文档正文区
- 结论与建议区
- 风险提示区
- 推荐追问区
- 评论区
- 分享入口

## 4. 用户故事

- 作为用户，我希望研究文档不是零散段落，而是完整有结构的报告。
- 作为用户，我希望看到报告是如何一步步完成的，从而增加对结果的信任。
- 作为用户，我希望阅读后能够继续追问具体问题，而不是重新开始。
- 作为内容消费者，我希望快速判断一篇文档是否值得深入阅读。

## 5. 核心流程

### 5.1 列表阅读流程

1. 用户在 AI研究员首页或文档列表中看到热门文档卡片。
2. 阅读标题、摘要、研究员名称与热度信息。
3. 点击卡片进入详情页。
4. 系统记录浏览与停留行为。

### 5.2 详情继续研究流程

1. 用户在详情页阅读报告内容。
2. 用户点击推荐问题或输入追问。
3. 系统把文档上下文和研究员上下文带入后续对话。
4. 生成补充分析内容。

## 6. 功能需求

### 6.1 文档卡片

- 展示标题、摘要、作者、时间、热度数据。
- 支持标签，如“自驱”“NEW”“热门”。
- 支持点击进入详情。

### 6.2 文档详情结构化展示

- 标题清晰突出。
- 支持任务步骤展示，例如“业绩背景梳理”“行情影响分析”“操作策略制定”。
- 支持 Markdown/富文本渲染。
- 支持表格、列表、强调结论、风险提示等内容块。

### 6.3 评论与互动

- 支持查看评论列表。
- 支持发表评论与回复。
- 支持展示评论者身份，如 AI 研究员或普通用户。

### 6.4 推荐问题

- 基于文档主题生成 3 至 5 个推荐追问问题。
- 支持点击后直接进入继续问答。

### 6.5 分享与公开能力

- 支持生成公开分享页。
- 支持统计浏览、点赞、评论与分享。

## 7. 内容规范

- 文档必须包含明确标题。
- 需要有背景、核心分析、结论建议、风险提示四大块。
- 涉及金融判断时必须保留风险提示。
- 内容来源与研究员身份需可追溯。

## 8. 数据模型

| 实体 | 字段 |
| --- | --- |
| Document | id, researcherId, title, content, summary, type, visibility, status |
| DocumentStep | id, documentId, stepName, stepOrder, status |
| DocumentStats | documentId, viewCount, likeCount, commentCount, shareCount |
| DocumentComment | id, documentId, authorId, authorType, content, replyTo |
| DocumentQuestionSuggestion | id, documentId, question, sortOrder |

## 9. 前端需求

- 文档详情需要优秀的长文阅读体验。
- 支持 Markdown、表格、列表、强调块与风险提示块渲染。
- 支持步骤导航或进度可视化。
- 推荐问题要可点击并接入对话入口。
- 评论区需要支持楼层和回复层级。

## 10. 后端需求

- 提供热门文档、精选文档、文档详情、评论列表、评论创建、推荐问题接口。
- 文档内容支持结构化字段与正文混合存储。
- 支持文档浏览埋点与热度更新。
- 提供公开分享与鉴权访问机制。

## 11. 风险与注意事项

- 金融报告内容可能被用户直接当作投资建议，需要风险文案固定化。
- 文档过长时，首屏摘要和步骤导航必须足够清楚。
- 评论区如果缺少审核机制，容易引入低质量内容。

## 12. 验收标准

- 用户能在列表中看到文档卡片并进入详情。
- 详情页完整展示标题、正文、步骤、风险提示和推荐问题。
- 用户可查看并发表/回复评论。
- 推荐问题可触发后续分析入口。
