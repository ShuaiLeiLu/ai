# 后端联调接口清单（v1）

基础前缀：`/api/v1`

## system
- `GET /health`
- `GET /live`

## auth
- `POST /auth/login`
- `POST /auth/register`
- `GET /auth/me`

## researchers
- `GET /researchers`
- `GET /researchers/{researcher_id}`
- `POST /researchers`
- `PATCH /researchers/{researcher_id}`
- `POST /researchers/{researcher_id}/hire`
- `POST /researchers/{researcher_id}/dismiss`
- `GET /researchers/market`
  - query: `q`、`page`、`page_size`
- `GET /researchers/market/{researcher_id}`
- `GET /researchers/mine`
- `POST /researchers/{researcher_id}/duplicate`
- `POST /researchers/{researcher_id}/publish`
- `POST /researchers/{researcher_id}/unpublish`
- `GET /researchers/options/skills`
- `GET /researchers/options/knowledge-bases`
- `GET /researchers/options/mcp-servers`
- `POST /researchers/{researcher_id}/test-chat`
  - body: `question`
  - `POST /researchers` body 支持：`name`、`title`、`style`、`description`、`prompt`、`visibility`、`skills`、`knowledge_bases`、`mcp_servers`、`self_drive_tasks`
  - `PATCH /researchers/{researcher_id}` body 支持：`title`、`style`、`description`、`prompt`、`visibility`、`skills`、`knowledge_bases`、`mcp_servers`、`self_drive_tasks`
  - 列表接口统一返回 `data.items` + `data.total`（market、mine、options）
