# 基础工程启动说明

## 1. 当前已完成的底层架构

### 前端

- Next.js 14 App Router
- React 18 + TypeScript
- Ant Design + Tailwind CSS
- Zustand
- TanStack Query
- React Hook Form + Zod 预留接入位
- ECharts 预留接入位

### 后端

- FastAPI
- SQLAlchemy 2.0
- Alembic
- PostgreSQL
- Redis
- Celery + Celery Beat
- SSE 预留封装
- OSS / S3 集成层

### 基础设施

- Docker Compose：Postgres / Redis / MinIO
- 根目录 Makefile
- 前后端环境变量模板

## 2. 推荐启动顺序

1. 复制根目录环境变量：`cp .env.example .env`
2. 启动基础设施：`make up`
3. 安装前端依赖：`make install-web`
4. 安装后端依赖：`make install-server`
5. 启动前端：`make dev-web`
6. 启动后端：`make dev-api`
7. 启动 Celery Worker：`make dev-worker`
8. 启动 Celery Beat：`make dev-beat`

## 3. 当前建议优先开发顺序

1. 认证与用户会话基础能力
2. 前端工作台壳层与导航壳层
3. 后端健康检查、配置中心、统一响应封装
4. AI研究员工作台首屏聚合接口
5. 人才市场与研究员编辑器
6. 文档中心与详情页
7. 任务编排、Webhook、模拟交易等异步能力

## 4. 当前骨架中的关键入口

- 前端首页：`web/app/(marketing)/page.tsx`
- 前端工作台：`web/app/(workstation)/workstation/page.tsx`
- 前端 AI研究员骨架页：`web/app/(workstation)/workstation/ai-researcher/page.tsx`

- 后端主路由：`server/app/api/v1/router.py`
- Celery 入口：`server/app/core/celery_app.py`
- 本地基础设施：`compose.yaml`
